## 📌 About The Project

**CINEMAX LK** is a Sinhala movie streaming website created to provide users with an easy way to watch Sinhala movies and TV series online.

This website is built with **HTML, CSS, JavaScript** and deployed using **Vercel**.

✨ Main goal:  
Provide a clean and fast platform to watch Sinhala entertainment content.
